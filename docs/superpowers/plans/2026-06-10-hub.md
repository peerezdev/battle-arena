# Hub (lobby) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Construir el Hub (lobby) presentacional con navegación real a los flujos existentes, y cambiar la landing ("Launch App" → Hub, sin Connect wallet).

**Architecture:** Componentes React puramente presentacionales bajo `src/ui/screens/Hub/`, compuestos por `Hub.tsx`, que recibe callbacks de navegación de `App.tsx`. Datos de chat/drops/battles/stats son MOCK (claramente no funcionales). Usa los tokens de `src/ui/theme.ts`. El motor y el backend no se tocan.

**Tech Stack:** React + TS + framer-motion + tokens de theme. Sin tests nuevos (UI presentacional): verificación = `tsc` + `vitest` (102 tests de motor) + `build` verdes, y revisión visual.

**Spec:** `docs/superpowers/specs/2026-06-10-hub-design.md`.
**Mockup de referencia (léelo):** `.superpowers/brainstorm/18398-1781618040/content/hub-v2.html`.

**Tokens disponibles** (`src/ui/theme.ts`): `COLORS` (bg #0a0d14-ish, panel, panel2, border, text, muted, green #14F195, violet #9945FF, red), `GRADIENT`, `SHADOW`, `FONTS.display`(Sora)/`mono`/`body`(Inter), `RARITY`, `formatUsd`. Usa `useReducedMotion` de `src/ui/useReducedMotion.ts` y el patrón `useIsWide` de `src/ui/components/BattleBoard.tsx` para responsive.

```tsx
import { useState } from 'react'
import { COLORS, GRADIENT, FONTS } from '../../theme'
import { useIsWide } from ... // copia el hook useIsWide de BattleBoard.tsx (o extráelo); usa '(min-width: 1100px)' para el dock y '(min-width: 760px)' para el rail
import { LeftRail } from './LeftRail'
import { QuickMatch } from './QuickMatch'
import { LiveBattles } from './LiveBattles'
import { ChatDock } from './ChatDock'
import { STAKE_OPTIONS, type HubNav, type LiveBattle } from './hubMockData'

interface Props {
  onPlayMana: () => void     // Mana Duel offline
  onPlayRoyale: () => void   // Battle Royale demo
  onOnchain: () => void      // Pack / Gacha / Create / Find match → flujo on-chain
}
export function Hub({ onPlayMana, onPlayRoyale, onOnchain }: Props) {
  const wideDock = useIsWide('(min-width: 1100px)')
  const wideRail = useIsWide('(min-width: 760px)')
  const [active, setActive] = useState<HubNav>('lobby')
  const [stake, setStake] = useState(STAKE_OPTIONS[1])
  const [chatOpen, setChatOpen] = useState(false)

  function go(id: HubNav) {
    if (id === 'mana') return onPlayMana()
    if (id === 'royale') return onPlayRoyale()
    if (id === 'pack' || id === 'gacha') return onOnchain()
    setActive(id) // lobby / ranks: queda en el hub
  }
  // LiveBattles.onSelectMode pasa 'pack'|'royale'|'gacha'|'mana' (subconjunto de HubNav) → go sirve igual.
  // Layout escritorio: grid rail(92) | main(1fr) | dock(340 si wideDock).
  // Top bar: título "Lobby · 18 players online", balance pill ($128.40 — EJEMPLO),
  // botón Deposit (onClick: no-op / "coming soon"), sonido.
  // Main scroll: <QuickMatch selectedStake={stake} onStake={setStake}
  //   onFindMatch={onOnchain} onCreate={onOnchain} />,
  //   <LiveBattles onSelectMode={go} onBattleAction={() => onOnchain()} />.
  // Dock: si wideDock, columna fija con <ChatDock/>. Si no, botón flotante 💬 que
  //   abre <ChatDock/> en un drawer (overlay) controlado por chatOpen.
  // Rail: si wideRail, <LeftRail active={active} onSelect={go}/> a la izquierda.
  //   Si no (móvil), una barra inferior fija con los mismos iconos (onSelect={go}).
  // Respeta useReducedMotion para cualquier animación de entrada/drawer.
}
```
